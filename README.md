# ECE-325

Labs and assignments for ECE 325 - OBJECT-ORIENTED SOFWARE DESIGN. Includes Labs 1–5 and the GuessGame project, covering OOP design, generics, collections, testing, and software engineering practices.

Made by Misbah — [MisbahAN.com](https://MisbahAN.com)

## Lab summaries

- `ece325_lab_assignment1`: Random phrase generator assets (`adjectives.txt`, `nouns.txt`), solution write-up, lab PDF, submission ZIP, and demo recording.
- `ece325_lab_assignment2`: Java generics and collections exercises with a solution PDF, lab brief, submission ZIP, and walkthrough video.
- `ece325_lab_assignment3`: Song ratings data set, design write-up, lab PDF, submission ZIP, and grading demo recording.
- `ece325_lab_assignment4`: Extended song ratings lab with solution assets, PDF spec, submission ZIP, and demo video.
- `ece325_lab_assignment5`: Testing-focused assignment with `testplan.pdf`, specification PDF, submission ZIP, and presentation recording.

## GuessGame

Self-contained Java implementation of the multi-player GuessGame along with a fairness verifier utility.
